# GDSQL Demos

Each demo is a Godot scene. Open and run in-editor.

| Demo | Scene | What It Shows |
|------|-------|---------------|
| Basic CRUD | `demo/basic_crud/demo_basic_crud.tscn` | INSERT, SELECT, UPDATE, DELETE |
| Advanced Queries | `demo/advanced_query/demo_advanced_query.tscn` | LEFT JOIN, GROUP BY, subqueries, UNION ALL |
| GBatis | `demo/gbatis/demo_gbatis.tscn` | XML mapper, entity class, method chaining |
