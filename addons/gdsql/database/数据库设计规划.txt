## 注：本文档最初设计时用，最新修改可能不会体现在本文档内，以实际情况为准
## 注：需兼容多人游戏，未来可能有在线联网、局域网、热座等模式

## 系统表
## t_system
## language: String				语言
## fullscreen: bool				是否全屏

## 玩家表
## t_user
## id: int						玩家id
## create_time: float			创建时间（unix_time）
## curr_level_id: int			当前所在关卡id

## 英雄表
## t_hero
## id: int						英雄id
## uid: int						所属玩家id
## tid: int						模板id
## level: int					等级
## exp: int						经验值
## max_hp: int					最大血量
## curr_hp: int					当前血量
## max_mp: int					最大魔法
## curr_mp: int					当前魔法
## max_ap: int					最大行动力
## curr_ap: int					当前行动力
## arr_ability: int				拥有天赋

## 技能表
## t_skill
## id: int								技能id
## uid: int								所属玩家id
## tid: int							模板id
## level: int							技能等级
## custom_deploy_range: Array[String]	自定义部署范围
## custom_deploy_center: Vector2		自定义部署范围对施放中心

## 装备表
# t_equipment
## id: int						装备id
## uid: int						所属玩家id
## tid: id						模板id
## level: int					装备等级

## 扇形表
## t_sector
## id: int						扇形id
## uid: int						所属玩家id
## tid: int					模板id
## angle: int					扇形角度（deg）
## skill_id: int				关联技能id（为0表示没关联）
## equiped: bool				是否装备到了转盘
## core_rotation: float			如果装备到了转盘上，sector_core的旋转位置
